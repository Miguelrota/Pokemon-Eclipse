
Proyecto realizado por Iago Fernandez, Miguel Rodríguez y Daniel García

Pokémon Eclipse es una aplicación de escritorio desarrollada en Java utilizando JavaFX 
que simula un combate por turnos entre dos Pokémon. El programa cuenta con dos 
escenas principales: un menú inicial y una pantalla de batalla. 
Desde el menú principal, el usuario puede iniciar una nueva partida o cargar una partida 
guardada previamente desde una base de datos MySQL. Al comenzar una partida, se 
accede a una segunda escena donde dos Pokémon se enfrentan. Cada uno de ellos tiene 4 
movimientos diferentes, con daños y usos limitados. El jugador puede seleccionar 
manualmente los ataques que desea realizar, mientras que el Pokémon rival responde con 
ataques aleatorios. 
Durante el combate, se puede guardar el estado actual de la partida (incluyendo la vida de 
los Pokémon y los usos restantes de cada movimiento) en una base de datos, permitiendo 
así retomarla más adelante.
